Product · System Design

Dynamic Scoring Engine

External judges needed to evaluate hundreds of applicants across multiple programmes, each with different criteria and scoring rubrics. The previous process was manual spreadsheets — slow, error prone, and impossible to scale.

Approach

Designed a configurable form engine where scoring logic, field weights, and evaluation rubrics are defined at the admin level. Forms are dynamically generated per programme. Judges receive role scoped access; scores aggregate in real time across all evaluators and surface in a results dashboard.

Admin Config → Form Schema Engine → Dynamic Forms → Score Aggregator → Results Dashboard
Node.js · Express · MySQL · React
~60% reduction in manual evaluation effort
Infrastructure · AWS

Legacy Platform to Modern AWS Stack

Inherited a codebase with no documentation, 15+ critical bugs, and recurring outages. The platform needed immediate stabilisation and a full migration to a maintainable, scalable architecture.

Approach

Phased migration to AWS — EC2 for compute, RDS for managed MySQL, S3 for assets, CloudFront for CDN, Route 53 for DNS, and SES with full DMARC/SPF/DKIM for transactional email. Nginx as reverse proxy with SSL termination. Redis for session and query caching. CloudWatch for monitoring and alerting.

CloudFront CDN → EC2 + Nginx → Node.js API → RDS MySQL
Route 53 SES · DMARC Redis S3 CloudWatch
Eliminated all recurring production outages
Product Feature

WYSIWYG Content Management System

Every content update required a developer — creating a 3 to 5 day turnaround for simple article changes and pulling engineering capacity away from real product work.

Approach

Built a rich text editor inside the admin panel with image uploads to S3, draft and publish workflow, SEO metadata fields, and a public facing rendering engine. Non technical staff now write, preview, and publish content without touching engineering.

70% reduction in content turnaround time
Infrastructure

Email Security and Deliverability

Transactional emails were landing in spam or failing to deliver entirely. No authentication was configured, damaging sender reputation at scale across a global audience.

Approach

Configured AWS SES with full DMARC, SPF, and DKIM. Set up a custom MAIL FROM domain for SPF alignment. Added CloudWatch bounce and complaint tracking. Graduated DMARC policy from none to quarantine after validating monitoring reports over several weeks.

Near 100% inbox delivery rate after configuration
